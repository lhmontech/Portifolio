Este é um Dashboard que desenvolvi enquanto estava aprendendo a usar o software Power Bi. Usei um arquivo CSV com dados de várias vendas de produtos, para alimentar o dashboard e criei gráficos que facilitam a ánalise dos dados. Dentre os gráficos presentes, temos um gráfico de colunas mostrando o faturamento por região, um gráfico de pizza mostrando o total de venda por produto, um gráfico de linha mostrando o faturamento por mês, cartões mostrando a quantidade total de vendas, o faturamento total e a média de venda por dia. Além disso, possui filtros por região, produto e data.

Essa é a aba inicial de uma API que desenvolvi em node.js + React para uma clínica de raio-x. A API consiste em apenas registrar todos os exames que são feitos, armazenando alguns dados do paciente, dados sobre qual exame foi realizado, quando foi e qual técnico realizou. Além da aba inicial onde é feito os registros, tem também abas para ver o histórico de todos os exames realizados e aba para extrair relatórios personalizados.
